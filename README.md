# 🛠️ Vakitzashi Toolbox

Коллекция полезных веб-приложений и инструментов для повседневных задач.

## 📋 Инструменты

- **🐕 Blind Dog** - Шифрование и дешифрование изображений
- **💱 Coin Converter** - Конвертер валют в реальном времени
- **🖼️ Image Processor** - Редактирование и обработка изображений
- **📄 Make PDF** - Преобразование изображений в PDF
- **🛂 Pass-Port** - Генератор паспортных данных
- **📦 PDF Compressor** - Сжатие PDF-файлов
- **🥗 Protein Counter** - Подсчет белков и калорий
- **📝 T2PDF** - Конвертация текста и Markdown в PDF
- **🔗 UTM Generator** - Генератор UTM-параметров для отслеживания

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере
2. Выберите нужный инструмент из главного меню
3. Наслаждайтесь функциональностью!

## 💻 Использование

Все приложения работают полностью в браузере без необходимости установки. Просто откройте HTML-файл и начните использовать.

### Локальный запуск

```bash
# Если у вас есть Python
python -m http.server 8000

# Или используйте любой другой локальный сервер
```

Затем откройте `http://localhost:8000` в браузере.
